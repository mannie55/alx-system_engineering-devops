# System Engineering & DevOps

This repository contains a collection of shell scripting exercises and projects completed as part of the ALX System Engineering & DevOps curriculum. Each directory corresponds to a specific topic or project, covering fundamental concepts of the shell, permissions, I/O redirection, and more.

## Directory Structure

- `0x00-shell_basics`: Introduction to basic shell commands and navigation.
- `0x01-shell_permissions`: Managing file and directory permissions.
- `0x02-shell_redirections`: Understanding and using I/O redirection and filters.
- `0x03-shell_variables_expansions`: Working with variables and expansions in the shell.

## Usage

The files in each directory are typically executable shell scripts that perform a specific task as described by their names.
